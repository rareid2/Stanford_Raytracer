\subsubsection*{template$<$typename Xpr\+Type, template$<$ class $>$ class Make\+Pointer\+\_\+$>$\newline
class Eigen\+::\+Tensor\+Forced\+Eval\+Op$<$ Xpr\+Type, Make\+Pointer\+\_\+ $>$}

\mbox{\hyperlink{class_eigen_1_1_tensor}{Tensor}} reshaping class. 

{\ttfamily template $<$class$>$ class Make\+Pointer\+\_\+} is added to convert the host pointer to the device pointer. It is added due to the fact that for our device compiler {\ttfamily T$\ast$} is not allowed. If we wanted to use the same Evaluator functions we have to convert that type to our pointer {\ttfamily T}. This is done through our {\ttfamily Make\+Pointer\+\_\+} class. By default the Type in the {\ttfamily Make\+Pointer\+\_\+$<$T$>$} is {\ttfamily T$\ast$} . Therefore, by adding the default value, we managed to convert the type and it does not break any existing code as its default value is {\ttfamily T$\ast$}.
