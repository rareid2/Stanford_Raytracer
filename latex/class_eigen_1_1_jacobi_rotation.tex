\doxysubsection{Detailed Description}
Rotation given by a cosine-\/sine pair. 

\textbackslash{}jacobi\+\_\+module

This class represents a Jacobi or Givens rotation. This is a 2D rotation in the plane {\ttfamily J} of angle $ \theta $ defined by its cosine {\ttfamily c} and sine {\ttfamily s} as follow\+: $ J = \left ( \begin{array}{cc} c & \overline s \\ -s & \overline c \end{array} \right ) $

You can apply the respective counter-\/clockwise rotation to a column vector {\ttfamily v} by applying its adjoint on the left\+: $ v = J^* v $ that translates to the following \mbox{\hyperlink{namespace_eigen}{Eigen}} code\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{v.applyOnTheLeft(J.adjoint());}
\end{DoxyCode}
