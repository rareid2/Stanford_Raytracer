\doxysubsection{Detailed Description}
A versatile sparse matrix representation where each element is a block. 

This class provides routines to manipulate block sparse matrices stored in a B\+S\+R-\/like representation. There are two main types \+:


\begin{DoxyEnumerate}
\item All blocks have the same number of rows and columns, called block size in the following. In this case, if this block size is known at compile time, it can be given as a template parameter like 
\begin{DoxyCode}{0}
\DoxyCodeLine{BlockSparseMatrix<Scalar, 3, ColMajor> bmat(b\_rows, b\_cols);}
\end{DoxyCode}
 Here, bmat is a b\+\_\+rows x b\+\_\+cols block sparse matrix where each coefficient is a 3x3 dense matrix. If the block size is fixed but will be given at runtime, 
\begin{DoxyCode}{0}
\DoxyCodeLine{BlockSparseMatrix<Scalar, Dynamic, ColMajor> bmat(b\_rows, b\_cols);}
\DoxyCodeLine{bmat.setBlockSize(block\_size);}
\end{DoxyCode}

\item The second case is for variable-\/block sparse matrices. Here each block has its own dimensions. The only restriction is that all the blocks in a row (resp. a column) should have the same number of rows (resp. of columns). It is thus required in this case to describe the layout of the matrix by calling set\+Block\+Layout(row\+Blocks, col\+Blocks).
\end{DoxyEnumerate}

In any of the previous case, the matrix can be filled by calling \mbox{\hyperlink{class_eigen_1_1_block_sparse_matrix_a81773c0b6563584956aaba80f915aff7}{set\+From\+Triplets()}}. A regular sparse matrix can be converted to a block sparse matrix and vice versa. It is obviously required to describe the block layout beforehand by calling either \mbox{\hyperlink{class_eigen_1_1_block_sparse_matrix_abde1d03830122a98f4d9d1336892ceb1}{set\+Block\+Size()}} for fixed-\/size blocks or set\+Block\+Layout for variable-\/size blocks.

Given a sparse matrix describing the nonzero block pattern, this function prepares the internal pointers for values. After calling this function, any {\itshape nonzero} block (bi, bj) can be set with a simple call to coeff\+Ref(bi,bj).
